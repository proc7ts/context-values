Enumeration CxRequestMethod

Context value request method.

Specifies how to obtain the value.

Assets

Assets: = 1
  • Defined in proc7ts/context-values/src/core/request-method.ts:41

Requests the value provided by context entry assets.

Calls CxEntry.Definition.assign method. If no value assigned, uses a fallback one if specified, or throws CxReferenceError otherwise.

When passed to CxRequest.set callback means the value is provided by entry assets.

Defaults

Defaults: = -1
  • Defined in proc7ts/context-values/src/core/request-method.ts:30

Requests the value by context entry defaults.

Calls CxEntry.Definition.assignDefault method. If no value assigned, uses a fallback one if specified, or throws CxReferenceError otherwise.

When passed to CxRequest.set callback means the value is a default one.

Fallback

Fallback: = 0
  • Defined in proc7ts/context-values/src/core/request-method.ts:19

Requests any available value.

The same as if no request method specified.

Calls CxEntry.Definition.assign method. If no value assigned, calls CxEntry.Definition.assignDefault one. If no value assigned in turn, uses a fallback one if specified, or throws CxReferenceError otherwise.

When passed to CxRequest.set callback means the assigned value is a fallback one.

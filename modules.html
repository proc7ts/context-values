CxGlobals

CxGlobals: CxValues
  • Defined in proc7ts/context-values/src/entries/globals.entry.ts:11

Global values context.

Contains values available globally for the whole application.

It is expected that there is exactly one global context exists per application.

CxGlobals

CxGlobals: CxEntry<CxValues, CxValues>
  • Defined in proc7ts/context-values/src/entries/globals.entry.ts:20

Context entry containing global values context.

Services may utilize this instance as a scope to make sure the service instance is singleton.

It is an application (or framework) responsibility to provide a global context.

cxScoped

  • cxScoped<TValue, TAsset, TContext>(scope: CxEntry<TContext, unknown>, definer: Definer<TValue, TAsset>): Definer<TValue, TAsset>
    • Defined in proc7ts/context-values/src/entries/scoped.entry.ts:19

    Creates scoped context value definer.

    Here "scope" is a context available as an entry of itself. Thus context derived from that scope may access it via the same entry.

    Scoped context value is the one available only in particular scope only.

    The entry definer created by this function accesses the scope, and then requests the value from that scope. Thus the value would be the same regardless the context it is requested from.

    Type parameters

    • TValue

    • TAsset = TValue

    • TContext: CxValues = CxValues

    Parameters

    • scope: CxEntry<TContext, unknown>

      Context entry containing target scope as its value.

    • definer: Definer<TValue, TAsset>

      An entry definer that defines the entry in target scope.

    Returns Definer<TValue, TAsset>

    New scoped context entry definer.
